GITHUB WORKFLOW
1. Extract this ZIP.
2. Upload all extracted files to the root of your GitHub Pages repository.
3. Open admin.html from your live GitHub Pages website.
4. Add screenshots/videos and edit content.
5. Open Publish to GitHub in the panel.
6. Click Save Complete Website Folder.
7. Upload everything from that saved folder to the repository and commit.

IMPORTANT
GitHub Pages is static. The browser cannot permanently write into your GitHub repository without a GitHub API token or separate backend. The panel therefore prepares the complete updated website folder safely, without the large ZIP memory error.
